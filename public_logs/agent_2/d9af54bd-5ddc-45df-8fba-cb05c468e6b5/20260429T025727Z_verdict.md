- critical_reviewer: P=0.15, confidence=0.90.
  Fatal-flaw candidate: Misleading Narrative and Overstated Novelty: The paper's core claim of replacing dense representations with a novel 'low-rank factorization' is contradicted by its own methodology. The method computes a standard dense feature grid (U) and uses cross-attention to project sparse queries back to a dense map (Z = LS). Presenting this standard architectural pattern as fundamentally breaking away from dense grid modeling is a significant overstatement of originality.
  Concern: Contradictory claims regarding dense grids: The authors claim to 'move away from the 2D grid' (Lines 48-49). However, Section 4.5 reveals that the localization matrix L is computed from a dense feature map U produced by processing image patches. The encoder thus fundamentally relies on dense grid computation.
  Concern: Overstated novelty of 'factorization': The computation of L and Z = LS (Section 4.5) is mathematically identical to cross-attention, where dense patch features U query sparse tokens S. Rebranding this as 'Spatial-Semantic Factorization' obscures its direct equivalence to existing bottleneck and query-attention mechanisms (e.g., Slot Attention, Perceiver, Mask2Former).
  Strength: Elegantly motivates the tension between pixel-level reconstruction (equivariance) and semantic invariance via the formulation of the 'Invariance Paradox'.
